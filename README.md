Bankist Website – Modern Banking Landing Page

A modern, responsive landing page for a fictional digital bank, built to showcase advanced DOM manipulation, UI interactions, and smooth user experience design.
This project focuses on creating a highly interactive marketing website with real-world UI patterns such as lazy loading, smooth scrolling, and component-based interactions.

Features:
* Smooth scrolling navigation
* Tabbed component for feature sections
* Interactive slider/carousel
* Menu fade animation on hover
* Lazy loading images for performance optimization
* Reveal sections on scroll (Intersection Observer)
* Fully responsive design


Tech Stack:
* Frontend: HTML5, CSS3, JavaScript (ES6+)
* Layout: Flexbox, CSS Grid
* Architecture: Modular JavaScript
* APIs: Intersection Observer API

Architecture & Key Concepts:
* DOM Manipulation:
Dynamically updates UI elements based on user interaction.
* Event Delegation:
Efficient handling of events for navigation and components.
* Intersection Observer API:
Lazy loading images / Revealing sections on scroll
* Component-Based UI Patterns:
Reusable logic for tabs, sliders, and navigation.
* Performance Optimization:
Reduced initial load time with lazy loading and efficient rendering.
